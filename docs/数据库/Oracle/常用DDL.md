# 创建用户

```sql
CREATE USER hsitsp  PROFILE DEFAULT IDENTIFIED BY hsitsp
DEFAULT TABLESPACE DATA TEMPORARY TABLESPACE TEMP ACCOUNT UNLOCK;
GRANT CONNECT TO hsitsp;
GRANT RESOURCE TO hsitsp;
GRANT CREATE ANY VIEW TO hsitsp;
GRANT CREATE DATABASE LINK TO hsitsp;
GRANT CREATE ANY SYNONYM TO hsitsp;
GRANT DEBUG CONNECT SESSION TO hsitsp;
GRANT CREATE JOB       TO hsitsp;
GRANT SCHEDULER_ADMIN  TO hsitsp;
GRANT MANAGE SCHEDULER TO hsitsp;
ALTER USER hsitsp QUOTA UNLIMITED ON DATA;
ALTER USER hsitsp QUOTA UNLIMITED ON INDX;
```
